Drive

Данные сервиса Drive экспортированы в следующих форматах:

  • Документы
    • DOCX – Microsoft Word (по умолчанию). Текстовые документы будут по умолчанию сохранены в формате Microsoft Word 2007 и выше.
    • ODF – текстовые файлы OpenDocument. С файлами, скачанными в этом формате, можно работать в открытых приложениях для обработки текста, например OpenOffice или LibreOffice.
    • RTF. Этот формат поддерживают большинство компьютеров. Однако в нем можно использовать не все способы форматирования, доступные в текстовых редакторах.
    • PDF. Документы, скачанные в этом формате, можно просматривать с помощью большинства современных операционных систем и браузеров, а также в программе Adobe Reader. Однако для изменения таких файлов нужно приобрести специальный редактор.
    • TXT. В файле этого формата сохранится только текст документа без учета стиля и форматирования.
  • Таблицы
    • XLSX – Microsoft Excel (по умолчанию). Таблицы будут по умолчанию сохранены в формате Microsoft Excel 2007 и выше.
    • ODS – таблицы OpenDocument. С файлами, скачанными в этом формате, можно работать в открытых приложениях для обработки текста, например OpenOffice или LibreOffice.
    • PDF. Таблицы, скачанные в этом формате, можно просматривать с помощью большинства современных операционных систем и браузеров, а также в программе Adobe Reader. Однако для изменения таких файлов нужно приобрести специальный редактор.
  • Презентации
    • PPTX – Microsoft PowerPoint (по умолчанию). Презентации будут по умолчанию сохранены в формате Microsoft PowerPoint 2007 и выше.
    • ODP – презентации OpenDocument. С файлами, скачанными в этом формате, можно работать в открытых приложениях для обработки текста, например OpenOffice или LibreOffice.
    • Изображение PNG. При экспорте презентации в виде изображений PNG на каждом из них будет отображаться отдельный слайд.
    • PDF. При экспорте в формат PDF каждый слайд будет отображаться на отдельной странице документа.
    • TXT. В файле этого формата сохранится только текст презентации без учета форматирования.
  • Формы
    • XLSX – Microsoft Excel (по умолчанию). Ответы на формы, созданные с помощью Google Диска, будут по умолчанию сохранены в формате Microsoft Excel 2007 и выше.
